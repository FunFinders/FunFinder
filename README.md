# FunFinder

A recommendation app that suggests activities to do in a new area.

# Run instructions

1. Install dependencies

   ```bash
   npm install
   ```

2. Start server in separate terminal
   ```bash
   python server.py
   ```

3. Start app in separate terminal
   ```bash
   cd funfinder-frontend # make sure in funfinder-frontend directory
   npx expo start
   ```
